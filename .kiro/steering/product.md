# 产品概述

## Kiro IES Generator

Kiro IES Generator 是一个基于 Blender Python API 的自动化工具，用于将 3D 灯具模型转化为符合 IESNA LM-63 标准的光度学文件（.ies）。

### 核心价值

通过 Cycles 物理渲染引擎模拟半透明材质（透射、散射、次表面散射）的光学特性，生成具有视觉真实感的渲染用 IES 文件。

### 主要功能

- **3D 模型导入**：支持 OBJ、FBX、STL 格式的灯具模型
- **物理材质模拟**：精确模拟磨砂玻璃、乳白亚克力、透明玻璃、纸质等材质的光学特性
- **球面采样**：基于 IESNA LM-63 C-Plane 坐标系进行精确的光强测量
- **IES 文件生成**：输出符合 LM-63-2002 标准的光度学数据文件

### 应用场景

- 灯具设计师：快速生成灯具的光度学数据用于产品展示
- 建筑可视化：为渲染引擎（V-Ray、Unreal Engine）提供真实的灯光数据
- 照明工程：验证灯具设计的光学性能
